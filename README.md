# 온라인 교육 사이트 프로젝트

그린컴퓨터 아카데미 수료 포트폴리오로 제출한 온라인 강의 사이트의 프론트엔드 코드입니다.

## 사용기술

### 프론트엔드

작성언어: TypeScript, HTML, CSS

프레임워크: Angular, Bootstrap CSS

### 백엔드

작성언어: Java

프레임워크: Spring Boot, Spring Data JPA, Spring Data REST

데이터베이스: MySQL
